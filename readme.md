## 使用說明 
### Setup 
專案使用`CocoaPods`版本`0.36`

安裝方式

```
sudo gem install cocoapods
```

打開專案後請移除`SecertKeys.swift`

並且註解掉`AppDelegate.swift`中之

```swift
Flurry.startSession(SecretKey.FlurryProductionKey)
Flurry.startSession(SecretKey.FlurryDevelopmentKey)
```

然後按下`cmd+R`就可以執行了。
